
# Hotel Booking Management System

A MERN Stack-based web application to handle hotel room reservations and admin management.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Setup Instructions

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Features
- View available rooms
- Book a room
- Admin: add new rooms
